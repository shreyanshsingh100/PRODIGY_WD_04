# PRODIGY_WD_04

Weather App

A dynamic weather application built using HTML, CSS, and JavaScript.

Overview

This Weather App provides real-time weather updates for locations worldwide. Built as an internship project for Prodigy InfoTech, this application demonstrates expertise in HTML, CSS, and JavaScript.

Features

- Dynamic weather updates
- Responsive design for seamless user experience
- Location-based search
- Current weather conditions (temperature, humidity, wind speed)
- 5-day weather forecast

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API

Usage

1. Enter a location in the search bar.
2. Click the "Get Weather" button.
3. View current and forecasted weather conditions.

API Documentation

This application utilizes the OpenWeatherMap API.
